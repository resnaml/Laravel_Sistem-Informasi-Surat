Untuk Database Bisa menggunakan DB seed caranya:
1. Buka folder aplikasi menggunakan VSCODE/text editor lainnya
2. Buka menu Terminal
3. Lalu Ketik "php artisan db:seed"
lalu Database terisi otomatis.

Untuk Mengisi data pada database seeder:
1. Buka folder aplikasi menggunakan VSCODE/text editor lainnya
2. Buka folder App\Database\sedeers
3. lalu masukan data yang anda ingin kan
